MyFlix API Documentation

All endpoints tested with Postman. JWT authentication required for most requests.

GET /movies

Authentication: Bearer JWT required (temporarily public)

Returns a list of all movies in the database.

Request Body: none
Response: JSON array of movie objects
Status Codes: 200 OK
Example URL: https://myflix-api-0vxe.onrender.com/movies

GET /movies/:title

Authentication: Bearer JWT required

Returns details of a specific movie by title.

Example URL: /movies/Inception
Request Body: none
Response: JSON object of the movie
Status Codes: 200 OK, 404 Not Found

GET /genres/:name

Authentication: Bearer JWT required

Returns information about a specific genre.

Example URL: /genres/Action
Request Body: none
Response: JSON object with genre details
Status Codes: 200 OK, 404 Not Found

GET /directors/:name

Authentication: Bearer JWT required

Returns information about a specific director.

Example URL: /directors/Christopher%20Nolan
Request Body: none
Response: JSON object with director details
Status Codes: 200 OK, 404 Not Found

POST /users

Authentication: none

Registers a new user.

Request Body:
{
  "Username": "mariobianchi",
  "Password": "Pass1234!",
  "Email": "mario.bianchi@example.com",
  "Birthday": "1990-01-01"
}
Response: JSON object of created user
Status Codes: 201 Created, 422 Validation Error
Example URL: https://myflix-api-0vxe.onrender.com/users

POST /login

Authentication: none

Login endpoint. Returns JWT token for authentication.

Request Body:
{
  "username": "mariobianchi",
  "password": "Pass1234!"
}
Response:
{
  "user": { ...userObject },
  "token": "JWT_TOKEN_HERE"
}
Status Codes: 200 OK, 400 Incorrect username or password
Example URL: https://myflix-api-0vxe.onrender.com/login

POST /users/:Username/movies/:MovieID

Authentication: Bearer JWT required

Adds a movie to the user's list of favorite movies.

Request Body: none
Response: JSON object of updated user
Status Codes: 200 OK, 404 Movie not found
Example URL: https://myflix-api-0vxe.onrender.com/users/mariobianchi/movies/68b71bf86d16a1626e8cf305

DELETE /users/:Username/movies/:MovieID

Authentication: Bearer JWT required

Removes a movie from the user's list of favorite movies.

Request Body: none
Response: JSON object of updated user
Status Codes: 200 OK, 404 Movie not found
Example URL: https://myflix-api-0vxe.onrender.com/users/mariobianchi/movies/68b71bf86d16a1626e8cf305

PUT /users/:Username

Authentication: Bearer JWT required

Updates a user's info (password, email, birthday).

Request Body:
{
  "Username": "mariobianchi",
  "Password": "NewPass123!",
  "Email": "newemail@example.com",
  "Birthday": "1990-01-01"
}
Response: JSON object of updated user
Status Codes: 200 OK, 400 Validation Error
Example URL: https://myflix-api-0vxe.onrender.com/users/mariobianchi

DELETE /users/:Username

Authentication: Bearer JWT required

Deletes a user account.

Request Body: none
Response: "Username was deleted."
Status Codes: 200 OK, 400 Permission denied, 404 User not found
Example URL: https://myflix-api-0vxe.onrender.com/users/testdelete
